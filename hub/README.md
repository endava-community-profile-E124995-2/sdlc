# Shared Hub

`hub/shared/` is the cross-project information layer for the workspace.

- `portfolio/`: workspace-level project inventory and executive summaries
- `standards/`: reusable policies, conventions, and guardrails
- `analytics/`: cross-project metrics and reporting
- `issues/`: shared issue patterns, recurring incidents, and watchlists
- `assets/`: reusable prompts, snippets, and Codex-facing support material
