# Execution Workflow

## Required Inputs

- approved architecture decision
- scoped work items
- dependency order and validation goals

## Role Handoffs

- primary: Architecture and Engineering Manager
- downstream handoff: Quality and Security and Governance Manager for verification planning

## Expected Artifacts

- engineering execution plan
- repo ownership map
- dependency sequence and validation checkpoints

## Output Location

- `projects/<project-id>/hub/technology/`
- `projects/<project-id>/hub/decisions/`
