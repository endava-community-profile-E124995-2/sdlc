# Context Variant: AI Product

## Use When

Use when probabilistic system behavior, evaluation quality, or human review expectations materially affect the recommendation.

## Overrides

- treat evaluation evidence as first-class release evidence
- separate prompt or model behavior claims from demonstrated evidence
- raise the bar for rollback and exception clarity when user impact is uncertain

## Risks To Emphasize

- `risk.verification_gap`
- `risk.evidence_gap`
