# Context Variant: Startup

## Use When

Use when the environment is fast-moving, low-process, and needs lightweight but explicit assurance.

## Overrides

- keep artifacts brief but concrete
- favor the smallest evidence set that can still support a real recommendation
- make ownership boundaries explicit when founders or small teams blend roles

## Risks To Emphasize

- `risk.verification_gap`
- `risk.evidence_gap`
