# Context Variant: Scale Up

## Use When

Use when coordination costs are rising and teams need clearer handoffs, evidence ownership, and release discipline.

## Overrides

- require clearer ownership on blockers and unblockers
- emphasize cross-team evidence gaps and remediation timing
- make exception handling and gate dependencies explicit

## Risks To Emphasize

- `risk.evidence_gap`
- `risk.gate_readiness_blind_spot`
