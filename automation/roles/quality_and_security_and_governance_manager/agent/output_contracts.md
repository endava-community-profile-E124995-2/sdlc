# Output Contracts

## Quality Review Note

- scope reviewed
- quality expectations
- major risks
- validation gaps
- recommendation

## Security Review Note

- boundary reviewed
- identified risks
- required follow-up
- escalation points

## Governance Mapping Note

- control or policy source
- affected behavior
- evidence or missing evidence
- approval questions

## Release Gate Memo

- gate scope
- pass or block recommendation
- blocking issues
- required handoffs
