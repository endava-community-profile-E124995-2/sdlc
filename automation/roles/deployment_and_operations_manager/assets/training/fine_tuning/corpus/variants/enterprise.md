# Context Variant: Enterprise

## Use When

Use when formal change windows, support tiers, auditability, and cross-functional approvals materially shape release work.

## Overrides

- raise the threshold for silent assumptions about rollback, runbooks, and receiving owners
- make checkpoint ownership and acceptance conditions explicit
- include formal communication and escalation expectations where they affect release readiness

## Risks To Emphasize

- `risk.rollback_readiness_gap`
- `risk.support_ownership_gap`
