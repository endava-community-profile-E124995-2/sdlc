# About This Role Package

This folder is the canonical Deployment and Operations Manager package for the workspace-first Codex SDLC model.

## Structure

- `agent/`: active docs Codex should read first
- `assets/training/fine_tuning/`: structured rollout and operations corpus, evals, schemas, and tooling
- `assets/knowledge/legacy_role_rag/`: fallback-only retrieval pack
- `builder/`: maintenance notes and migration plans for humans
- `integrations/`: mapping to the repo-local skill adapter
